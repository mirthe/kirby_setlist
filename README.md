# Kirby Plugin: Setlist.fm

This plugin allows you to include concert setlists from the Setlist.fm API on your pages.

## Example 

Add the following to your config:

    'setlistfm.apiKey' => 'XX',
       
Placed for example with 

    (setlist: show: 398b9b7)
